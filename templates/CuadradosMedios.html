¿Qué es el algoritmo de cuadrdos medios




Un número pseudo aleatorio es un número que cumple las funciones de un número aleatorio pero es generado a través de un algoritmo matemático. Si conocemos el algoritmo que se utilizó para generar una secuencia de números pseudo aleatorios, así como algunos valores iniciales, se puede predecir toda la secuencia de números que se van a generar.
